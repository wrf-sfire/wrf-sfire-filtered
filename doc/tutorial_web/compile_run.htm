Environment WPS & ARW WRF-VAR Feedback 

Compile and Run WRF ARW and WPS
 

Untar the code

tar -xf /wrfhelp/SOURCE_CODE/WPS+WRFV2.tar
 

Set up environment variable

The classroom computers has been set up so that it knowns where the netcdf libraries are located. This may be something you will have to explicity set for your home computer. Typically this will be something like:

     setenv NETCDF /usr/local/netcdf
 

Compile WRFV2

Even though one will run WPS first, compilation of this code depends on the successful compilation of the WRFV2 code.

cd WRFV2

./configure (Pick option 2 from the list)

./compile (for ARW real cases pick em_real)

./compile em_real >& make.log & (potential problem)

Check make.log to make sure your compile was successful.

 

Compile WPS

cd ../WPS

./configure (Pick option 1 from the list)

./compile >& make.log & (potential problem)

Check make.log to make sure your compile was successful.

 

If both WRFV2 and WPS compiled correctly, you are ready to set up and run the case(s) of your choice.
