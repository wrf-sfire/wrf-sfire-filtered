Libraries

To install "ungrib" (part of WPS) on your home computer, you will need the libraries, JasPer, PNG and Zlib. Zlib may already be on your computer, so check before installing this library. JasPer and PNG are compression libraries needed to ungrib GRIB version 2 data.

WPS input data

You need to have all the general input files, plus one or more of the resolution specific input file (10m, 5m, 2m, 30s)
