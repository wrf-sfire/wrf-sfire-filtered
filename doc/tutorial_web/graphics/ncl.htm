  Environment WPS & ARW WRF-VAR Feedback 

Graphics: NCL
 

Get the tar file:

tar –xvf /wrfhelp/SOURCE_CODE/WRF_NCL.tar
 

Environment variables:

setenv  NCARG_ROOT /usr/local/ncl
 

Create library:

WRAPIT wrf_user_fortran_util_0.f
   

Copy resource file:

            cp DOThluresfile ~/.hluresfile
 

Run NCL:

Edit one of the “*.ncl” script.
For real time cases, a good starting script is wrf_real.ncl (initially at least edit the path to your wrfout dataset.)

            ncl wrf_real.ncl
