  Environment WPS & ARW WRF-VAR Feedback 

Graphics: GrADS
 

Get the tar file:

tar –xvf  /wrfhelp/WRF2GrADS.tar
 

Environment variable:

setenv  GADDIR  /usr/local/grads/data
 

Compile:

make

(Tip: Once back at your home institute, you will need to edit the Makefile to pick your computer type and to make sure the netCDF paths are set correctly.)
 

Run the converter:

Edit control_file (initially at least set the time information, and the paths to your wrfout input data)
./wrf_to_grads  control_file  MyOutput
