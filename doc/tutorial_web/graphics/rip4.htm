  Environment WPS & ARW WRF-VAR Feedback 

Graphics: RIP4
 

Get the tar file:

tar –xvf  /wrfhelp/SOURCE_CODE/RIP4.tar
 

Environment variables:

setenv  RIP_ROOT  your-rip4-directory
 setenv  NCARG_ROOT /usr/local/ncarg
 

Compile:

make linux

(Tip: Once back at your home institute, you may need to make sure the netCDF paths are set correctly in the Makefile before your compile.)
 

Run ripdp_wrf:

mkdir TUTDATA
./ripdp_wrf  TUTDATA/myCase  all  WorkingDirectory/WRFV2/run/wrfout*
 

Run rip:

Edit rip_sample.in (initially at least set the time information)
./rip  -f  TUTDATA/myCase  rip_sample
