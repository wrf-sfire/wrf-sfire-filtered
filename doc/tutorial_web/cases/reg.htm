Adding a Variable to the history file
 

  • First view the history file (wrfout) and try to locate the variable "UST" (It should not be in the file by default)
     
     
  • Edit the file Registry/Registry.EM, and look for UST (frictional velocity). Change the current line that looks something like

     state   real   UST   ij   misc   1  -  r   "UST" "U* IN SIMILARITY THEORY"

to

     state   real   UST   ij   misc   1  -  rh   "UST" "U* IN SIMILARITY THEORY"

  • Type 'clean -a', 'configure' and 'compile em_real'
     
     
  • Run the model as before
     
     
  • Now view the history file (wrfout) and try tolocate the variable "UST" (It should now appear in your output file)
     

If this was successful, you can continue to run another case of your choice, or generate graphics for your case.
